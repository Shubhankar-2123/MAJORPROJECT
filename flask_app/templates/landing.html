Dataset & Training

Built on real-world ISL data for accurate and reliable translation

Image-Based

Static Gestures

  • ✓ Coverage: Indian Sign Language Alphabets & Numbers
  • ✓ Data: 1,000+ samples per gesture
  • ✓ Input: Hand landmark coordinates (MediaPipe)
  • ✓ Model: Neural Network (MLP)
  • ✓ Advantage: High accuracy & fast inference (<50ms)
🎬
Video-Based

Dynamic Sequences

  • ✓ Coverage: Words, Phrases & Sentences
  • ✓ Data: ~5 videos per word (custom recorded)
  • ✓ Input: Video frames & sequences
  • ✓ Model: CNN + LSTM
  • ✓ Advantage: Captures motion & temporal patterns

How It Works

🧠 Machine Learning Models

SignAI uses advanced neural networks to understand and generate sign language:

  • ✓ MLP Networks: For static sign recognition (letters, numbers)
  • ✓ LSTM Networks: For dynamic sign sequences (words, sentences)
  • ✓ Keyframe Extraction: MediaPipe-based hand/pose tracking
  • ✓ Confidence Scoring: Real-time prediction confidence visualization

📊 Dataset & Training

Trained on comprehensive sign language datasets:

  • ✓ Coverage: 220+ ISL gestures (alphabets, numbers, words, and sentences)
  • ✓ Data Augmentation: Enhanced with rotations and noise
  • ✓ Multi-user Training: Works with diverse signing styles
  • ✓ Continuous Learning: Feedback system for improvements
